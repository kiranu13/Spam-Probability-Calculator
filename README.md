# Spam Probability Calculator

## Overview

Spam Probability Calculator is a Flask-based web application that uses Machine Learning to classify messages as **Spam** or **Ham (Not Spam)**. It provides an easy-to-use interface where users can enter a message and instantly receive a prediction.

## Features

- Spam Detection using Machine Learning
- User-friendly Web Interface
- Message History
- SQLite Database Integration
- Fast Prediction System

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- SQLite
- Scikit-learn
- Pandas
- NumPy
